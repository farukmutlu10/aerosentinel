## 🔤 Tipografi

### Logo Fontu
- **Font:** Psilograph
- **Kullanım:** Sadece "AEROSENTINEL" logosu için
- **Weight 100:** "AERO" kısmı
- **Weight 600:** "SENTINEL" kısmı
- **Dosyalar:** `fonts/Psilograph-Thin.ttf`, `fonts/Psilograph-SemiBold.ttf`

### UI Fontu
- **Font:** Inter
- **Kullanım:** Tüm arayüz metinleri
- **Weights:** 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

## 🖼️ Logo Kullanım Kuralları

1. **Minimum boyut:** Logo en az 24px yükseklikte okunabilir olmalı
2. **Boşluk:** Logo etrafında minimum 16px padding olmalı
3. **Arka plan:** Koyu arka planlarda gold veya white monokrom, açık arka planlarda gold veya dark monokrom kullanılmalı
4. **Bozma:** Logo oranları değiştirilmemeli, döndürülmemeli, gölge eklenmemeli
